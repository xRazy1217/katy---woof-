# 🎨 Katy & Woof Creative Studio

## 📋 Sistema Modular v7.0 - Arquitectura Refactorizada

**Estado:** ✅ **Producción Lista** | **Fecha:** Marzo 2026

Sistema de gestión de portafolio artístico completamente modular, optimizado para mantenibilidad y escalabilidad.

---

## 🚀 Inicio Rápido

### 1. Configuración del Entorno
```bash
# Instalar dependencias
npm install

# Configurar base de datos en config.php
# Ejecutar esquema inicial
mysql -u username -p database < INIT_SCHEMA_SYSTEM.sql
```

### 2. Acceso al Sistema
- **Sitio Web:** Abrir `index.php`
- **Panel Admin:** Abrir `admin.html`
- **Validación:** Ejecutar `node validate-system.js`

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
